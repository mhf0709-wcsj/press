# 数据库集合初始化指南

系统检测到您当前运行环境缺少必要的数据库集合，导致“设备台账”和“生命周期”功能无法正常写入数据。请按照以下步骤手动初始化：

## 1. 打开云开发控制台
在微信开发者工具中，点击工具栏上的 **“云开发”** 按钮。

## 2. 进入数据库管理
点击顶部的 **“数据库”** 标签页。

## 3. 创建集合
请依次点击 **“+”** 号，创建以下集合（Collection Name）：

| 集合名称 | 说明 |
| :--- | :--- |
| **`equipments`** | 设备库（母体设备），一个设备可绑定多个压力表 |
| **`devices`** | 压力表台账（一表一码），每个压力表必须关联一个设备 |
| **`lifecycle_logs`** | 压力表生命周期流转轨迹（入库、安装、送检等） |
| **`pressure_records`** | 压力表检定记录（拍照识别入库的数据） |
| **`enforcement_cases`** | 监管现场核验取证案件（可选：监管端功能） |
| **`evidence_items`** | 取证照片明细（可选：监管端功能） |
| **`kb_docs`** | 合规知识库文档（可选：RAG 合规助手） |
| **`kb_chunks`** | 合规知识库分片与向量（可选：RAG 合规助手） |

> **注意**：集合名称必须完全一致（区分大小写，不要有多余空格）。

## 4. 设置权限（可选）
默认权限通常为“仅创建者可读写”，建议根据实际需要调整为“所有用户可读，仅创建者可写”或自定义安全规则，以确保多用户协作时的权限正常。

完成以上步骤后，请**重新编译**小程序，错误提示即可消失。
