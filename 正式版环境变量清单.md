# 正式版环境变量清单

适用项目：压力表检定智能体小程序

---

## 1. 必配云函数环境变量

### `aiAssistant`

- `DASHSCOPE_API_KEY`
  - 用途：阿里百炼 / 通义千问大模型问答
- `DASHSCOPE_MODEL`
  - 用途：指定问答模型
  - 建议值：`qwen3.5-flash`
- `DASHSCOPE_ENDPOINT`
  - 用途：百炼兼容接口地址
  - 建议值：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
  - 可选，不配时代码内有默认值

### `baiduOcr`

- `BAIDU_API_KEY`
  - 用途：百度 OCR API Key
- `BAIDU_SECRET_KEY`
  - 用途：百度 OCR Secret Key

---

## 2. 建议配置的环境变量

### `expiryReminder`

- `DEVICE_EXPIRY_TEMPLATE_ID`
  - 用途：设备到期微信订阅消息模板 ID
  - 不配置时，到期提醒统计还能用，但不会实际发送订阅消息

---

## 3. 每个云函数建议配置

### `aiAssistant`

- [ ] `DASHSCOPE_API_KEY`
- [ ] `DASHSCOPE_MODEL`
- [ ] 可选补 `DASHSCOPE_ENDPOINT`
- [ ] 超时时间建议调到 `20-30 秒`

### `baiduOcr`

- [ ] `BAIDU_API_KEY`
- [ ] `BAIDU_SECRET_KEY`
- [ ] 超时时间建议不少于 `20 秒`

### `enterpriseAuth`

- [ ] 当前无额外第三方密钥要求
- [ ] 确认和正式数据库环境绑定一致

### `expiryReminder`

- [ ] `DEVICE_EXPIRY_TEMPLATE_ID`
- [ ] 确认微信订阅消息模板已在对应环境可用

---

## 4. 配置位置

在微信开发者工具中操作：

1. 打开 `云开发`
2. 选择正式使用的环境
3. 进入 `云函数`
4. 点开具体云函数
5. 找到 `环境变量`
6. 新增键和值
7. 保存后重新部署该云函数

---

## 5. 配置完成后要马上验证

- [ ] `aiAssistant` 问开放问题能返回大模型回答
- [ ] `baiduOcr` 上传图片能正常识别
- [ ] `enterpriseAuth` 微信登录与绑定企业正常
- [ ] `expiryReminder` 获取摘要和提醒接口正常

---

## 6. 安全要求

- [ ] 不要再把 Key 写回代码
- [ ] 已暴露过的旧 Key 立即作废并重建
- [ ] 开发环境和正式环境使用不同的 Key
- [ ] 只给需要的云函数配置对应变量，不要全环境共用一套
